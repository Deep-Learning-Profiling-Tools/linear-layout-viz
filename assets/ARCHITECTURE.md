The `assets/` directory contains source media used by the README and project pages.

These files are not application inputs. They are screenshots and short clips that help a visitor understand what LL-viz looks like before running it. Keep media filenames tied to the workflow they demonstrate, such as preset loading, selection, coloring, or SVG export.

When UI behavior changes enough that a clip becomes misleading, update the clip in the same change as the documentation that references it.
