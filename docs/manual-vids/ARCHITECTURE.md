The `docs/manual-vids/` directory contains short videos referenced by the manual.

Each file demonstrates one user workflow: dragging, selection, inspector behavior, matrix view, tabs, slicing, coloring, cell text, display toggles, or saving. The videos should stay small and focused so reviewers can tell which workflow changed.

When a workflow changes, update the corresponding manual text and only the affected clip. Avoid replacing unrelated clips because that makes documentation reviews harder.
